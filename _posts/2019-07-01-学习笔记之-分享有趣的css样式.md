## word-wrap的属性
word-wrap属性允许长的内容可以自动换行。它设置或检索当内容超出其容器边界时是否中断单词。
* 例如，加入css样式

```
<!DOCTYPE html>
<html>
<head>
<style> 
p{
width:11em; 
border:1px solid #000000;
}
p.test{
word-wrap:break-word;
}
</style>
</head>
<body>
<p>dsffdgrh121324gfdgfhhgjhkkjlkl;</p>
<p class="test">dsffdgrh121324gfdgfhhgjhkkjlkl;</p>
</body>
</html>
```
